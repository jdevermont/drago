DRAGO Platform - Leader Photos
================================

Add your leader photos to this folder with the following filenames:

CURRENT LEADERS:
- matlanyane.jpg    (Retšelisitsoe Matlanyane - Lesotho Finance Minister)
- ramaphosa.jpg     (Cyril Ramaphosa - South Africa President)

PHOTO SPECIFICATIONS:
- Recommended size: 400x400 pixels (square)
- Minimum size: 200x200 pixels
- Format: JPG or PNG
- The photos will be displayed as squares with rounded corners

ADDING NEW LEADERS:
When you add a new leader to the platform, add their photo here with a
matching filename. For example, if you add a leader with id: 'biden',
name the photo: biden.jpg

In the JavaScript code, set:
  photo: 'images/biden.jpg',

FILE STRUCTURE:
your-folder/
  ├── drago-platform.html
  └── images/
      ├── matlanyane.jpg
      ├── ramaphosa.jpg
      └── [other-leader].jpg
